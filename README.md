# PhotoMosaicMaker

비디오/이미지에서 추출한 소스 이미지로 포토 모자이크(Photomosaic) 이미지를 생성하는 로컬 WPF 애플리케이션입니다.

타겟 이미지의 색/구성을 타일 단위로 분석하고, 가장 유사한 소스 이미지를 매칭하여 하나의 큰 결과 이미지를 구성합니다.

### 프로젝트 목표

대용량 이미지 처리 파이프라인(추출 → 정제 → 라이브러리 빌드 → 렌더) 설계/구현 경험

WPF(MVVM) 기반 UI와 Core 라이브러리 분리로 확장 가능한 구조를 만드는 데 집중

실제 사용을 전제로 한 입출력/경로 처리, 진행률 표시, 취소, 캐시 관리 등 완성도를 목표로 개선중

### 주요 기능

타겟 이미지 → 모자이크 렌더링

타일 크기 설정

결과 미리보기(화면에 맞게 자동 확대/축소)

로컬 비디오 파일 추가

유튜브 URL만으로 동영상 파일 등록

프레임 추출(FPS 및 최대 프레임 제한)

동영상 소스 수집 (샘플링)

dHash 기반 유사 이미지 제거(중복 프레임 삭제)

이미지 데이터 입력/변경/출력/저장

### 프로젝트 구조

PhotoMosaicMaker.App : WPF UI (MVVM)

PhotoMosaicMaker.Core : 모자이크 엔진/이미지 처리 로직

PhotoMosaicMaker.TestCLI : 핵심 로직 검증용

### 사용 기술

C# / .NET 8

WPF (MVVM)

SixLabors.ImageSharp (이미지 처리)

ffmpeg (프레임 추출)

yt-dlp (URL 기반 동영상 다운로드)

### 실행/사용 흐름

Target Image 선택

→ Output Folder 확인(자동 생성)

→ Video Sources에 파일 또는 URL 추가

→ Extract Frames 실행 (동영상 샘플링) → 소스 폴더에 프레임 생성 및 유사 프레임 정제

→ Build Library 실행 (타일 데이터 로드)

→ Render 실행 → 결과 이미지 저장 및 미리보기 확인

##### * 선택사항

초당 동영상 샘플링 횟수

모자이크 타일 크기

### 라이선스/주의사항

본 프로젝트는 개인 포트폴리오 목적의 실험/학습용으로 제작되었습니다.

외부 도구(ffmpeg/yt-dlp)는 각 도구의 라이선스를 따르며, YouTube 등 서비스의 이용약관을 준수하여 사용해야 합니다.
